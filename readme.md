# AI ML Stock Market Analyzer

## Project Overview
The AI ML Stock Market Analyzer is a Python-based project developed using Streamlit, Yahoo Finance API, and Machine Learning concepts.

The main purpose of this project is to help users analyze stock market trends in an easy and interactive way. The application allows the user to enter any stock ticker symbol and visualize important stock indicators along with a simple next-day closing price prediction.

This project solves the real-world problem of understanding stock movement patterns and performing basic forecasting using historical data.

---

## Problem Statement
Investors and students often find it difficult to understand stock trends, technical indicators, and future price movement.

This project provides:
- stock trend visualization
- moving average analysis
- RSI and MACD indicators
- risk metric calculation
- next-day stock price prediction

This makes stock analysis easier for learning and educational purposes.

## Installation and Setup
### Step 1: Clone or download the project
Download the project folder or clone the GitHub repository.

### Step 2: Install required libraries
Run the following command in terminal:

pip install -r requirements.txt

### Step 3: Run the project
Use the following command:

streamlit run app.py

### Step 4: Open in browser
After running the above command, Streamlit will generate a local URL:

http://localhost:8501

Open this URL in your browser.

## How to Use
1. Open the application in browser
2. Enter stock ticker symbol such as:
   - AAPL

## Machine Learning Workflow
The project uses **Linear Regression** for next-day stock closing price prediction.

### Input Features Used
- previous 1 day closing price
- previous 2 day closing price
- previous 3 day closing price
- MA 10
- MA 20
- MA 50
- volatility
- RSI
- MACD
- MACD signal

### Output
- next-day predicted closing price

The model is trained on 80 percent of historical data and tested on the remaining 20 percent.
